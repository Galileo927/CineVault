# Roadmap

## Phase 1: MVP

- Project skeleton
- Drogon server startup
- SQLite schema
- Library directory configuration
- File scanning
- FFprobe media parsing
- Movie list API
- Poster wall
- HTTP Range playback
- Playback progress save and resume
- Same-name SRT subtitle matching

## Phase 2: Product Usability

- Incremental scan
- Search, recently watched, favorites, watched/unwatched
- Metadata editing
- ASS subtitles
- Subtitle switching and delay
- Better error diagnosis
- Local startup polish

## Phase 3: Small Commercial Build

- Remote access guide
- HLS transcoding and cache
- Multi-user history and favorites
- Poster wall performance tuning
- Mobile layout
- Backup and restore
