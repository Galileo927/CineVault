# UI Principles

CineVault should feel like a private movie library, not an admin panel.

## Home

- Lead with continue watching, recently added, favorites, and a poster wall.
- Keep filesystem paths and technical metadata secondary.
- Prioritize poster loading and scroll performance.

## Detail Page

- Treat a movie as a collectible object.
- Show poster, backdrop, title, year, runtime, genre, overview, subtitles, and media quality tags.

## Player

- Controls should be quiet and predictable.
- Save progress on exit and during playback.
- Give clear reasons when direct playback is not available.
